# loja-suplementos
Loja virtual de suplementos desenvolvida com TypeScript, HTML e CSS, com catálogo de produtos, pesquisa em tempo real e integração com WhatsApp.

## Funcionalidades

- Catálogo com 14 suplementos
- Pesquisa de produtos em tempo real
- Exibição de imagens, descrição e preço
- Compra via WhatsApp
- Layout responsivo para dispositivos móveis
- Interface moderna com HTML e CSS

## Tecnologias Utilizadas

- TypeScript
- HTML5
- CSS3

## Objetivo

Projeto criado para aprimorar conhecimentos em TypeScript, manipulação do DOM, módulos ES6 e desenvolvimento de interfaces web.
